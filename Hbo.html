Leren coderen

Waarom zou je willen leren programmeren?

Leren coderen geeft je een nieuwe vaardigheid die niet veel andere bedrijfskundige zullen hebben. Ik ben ervan overtuigd dat de online marketing en het online shoppen nog grote stappen gaat maken. Denk aan apps en computersystemen van en voor bedrijven. Maar ook aan stofzuigrobots, navigatiesystem, en online webshops, dit alles heeft codering nodig om te weten wat het doet en wat het moet gaan doen. Coderen is dan ook de taal van de toekomst voor vrijwel alle bedrijven. Natuurlijk kan ik niet binnen 24 uur een professional worden in het vak van coderen. Wat ik wel kan doen is alvast de basis begrijpen zodat ik ook in wat ingewikkeldere bedrijven kan meepraten over de marketing, producten en de online webshops. Ik heb ervoor gekozen om de basis te leren voor HTML en CSS.

Wat is HTML en waarom deze taal?

HTML is een programmeertaal waar de meeste websites op gebouwd zijn of naar vervormd zijn. Dit is dus een belangrijke basis in het maken van een website/shop. Er zijn nog heel veel andere coderingstalen zoals JAVASCRIPT, Ook de Avans website maakt gebruik van HTML.

Wat is CSS?

CSS is een programmeertaal waarmee je de opmaak van een website maakt. Zo kan je kleuren, lettertypes en letterstijlen per letter opmaken.

Welke programma's heb ik gebruikt?

ATOM

Binnen ATOM kan je de verschillende bestanden die je nodig hebt openen en coderingen uittypen. Dit is een beetje de WORD van het coderen. Dus zowel het HTML als het CSS bestand kan hier worden bewerkt

Github

Op Github sla je je bestanden op in de cloud, op deze manier kunnen ze niet verloren gaan. Ook wordt GITHUB gebruikt om bestanden te sturen en te ontvangen wanneer je met meerdere mensen aan het werken bent in hetzelfde bestand.

MAMP

Met MAMP kan je de website voor een DEMO open zetten op je browser, zo krijg je een preview van hoe de website eruit komt te zien voor de bezoekers. Ook zou je de officiele website via deze manier openbaar kunnen maken, maar dan moet je een betaalde versie hebben.
